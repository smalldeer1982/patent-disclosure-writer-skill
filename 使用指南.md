# 专利交底书技能 - 使用指南

## 项目概述

这是一个 **Claude Code 技能**，用于自动化生成符合中国 IP-JL-027 标准的专利申请技术交底书。

**版本**: 2.2.0 | **最后更新**: 2025-01-16

## 核心功能

- 自动专利检索和技术调研
- 智能分析创新点和技术方案
- 自动生成 Mermaid 格式专业附图
- 支持断点续传和选择性重新生成
- 输出 Markdown 和 DOCX 格式
- 可选 5 人专家团队审核系统

## 快速开始

### 方式一：安装脚本（推荐）

使用安装脚本将技能安装到你的专利项目目录：

```bash
# 1. 创建专利项目目录
mkdir "我的专利项目"
cd "我的专利项目"

# 2. 运行安装脚本
python C:\WorkSpace\agent\patent-disclosure-writer-skill\install_patent_skill.py

# 按提示选择 [1] 安装技能
# 然后选择目标目录（如当前目录）
```

安装后在项目目录中：

```bash
# 启动 Claude
claude

# 生成专利交底书
/patent

# 导出黑白附图（退出 Claude 后）
python .claude/skills/patent-disclosure-writer/scripts/export_figures.py --dir . --pattern "0[3-7]_*.md"
```

### 方式二：使用 --plugin-dir

直接使用技能源目录作为插件：

```bash
claude --dangerously-skip-permissions --plugin-dir C:\WorkSpace\agent\patent-disclosure-writer-skill
```

**注意**：此方式可能存在脚本路径问题，建议使用安装脚本方式。

## 斜杠命令

| 命令 | 功能 |
|------|------|
| `/patent` | 智能生成交底书（支持断点续传、选择性重新生成） |
| `/patent-update-diagrams` | 智能补充附图（扫描章节并补充缺失的附图） |
| `/patent-export-figures` | 导出黑白附图（将 Mermaid 附图导出为黑白 PNG 图片） |
| `/patent-md-2-docx` | Markdown 转 DOCX（将 Markdown 交底书转换为正式格式） |

## 使用示例

### 基本使用

```
你：帮我写专利交底书

Claude：好的，我需要了解一些信息：
1. 你的创新想法是什么？
2. 所属技术领域是什么？
3. 有什么关键词可以帮助搜索吗？（可选）

你：
- 创新想法：一种基于深度学习的图像去雾方法，通过注意力机制增强去雾效果
- 技术领域：计算机视觉/图像处理
- 关键词：图像去雾、深度学习、注意力机制、卷积神经网络

Claude：开始生成专利交底书...
[自动执行多个子代理任务]

完成！交底书已生成：
- Markdown: 专利申请技术交底书_一种基于深度学习的图像去雾方法.md
- 各章节文件: 01_发明名称.md 到 09_参考资料.md
```

### 断点续传

如果生成过程中断，再次运行 `/patent` 命令时，技能会自动检测已完成的章节：

```
检测到已有 5 个章节文件：
✓ 01_发明名称.md
✓ 02_所属技术领域.md
✓ 03_相关的背景技术.md
✓ 04_解决的技术问题.md
✓ 05_技术方案.md

是否从已有章节继续生成交底书？
[继续执行] [重新生成] [选择特定章节重新生成]
```

## MCP 服务配置

本技能需要配置以下 MCP 服务（在 `~/.claude/settings.json` 中）：

| 服务 | 用途 | 获取地址 |
|------|------|----------|
| web-search-prime | 网络搜索 | https://open.bigmodel.cn/ |
| web-reader | 网页内容提取 | https://open.bigmodel.cn/ |
| google-patents-mcp | 专利检索 | https://serpapi.com/ |
| exa | 技术文档搜索 | https://exa.ai/api-key |

详细配置步骤见：[MCP 配置指南](.claude/skills/patent-disclosure-writer/references/configuration.md)

## 专利类型

| 类型 | 创新要求 | 审查周期 | 保护期限 |
|------|---------|---------|----------|
| 发明专利 | 突出的实质性特点和显著的进步 | 2-3年 | 20年 |
| 实用新型专利 | 实质性特点和进步 | 6-12个月 | 10年 |

## 审核系统（可选）

本技能支持可选的 5 人专家团队审核：

| 专家角色 | 权重 | 职责 |
|---------|------|------|
| 资深技术专家 | 2票 | 技术可行性、创新性 |
| 技术专家 | 1票 | 技术细节、实施方案 |
| 法律专家 | 1票 | 法律合规性 |
| 资深专利代理人 | 2票 | 保护范围、专利策略 |
| 专利代理人 | 1票 | 文档格式、规范性 |

## 常见问题

### Q: 安装后找不到 export_figures.py？

A: 确保使用安装脚本安装，而不是 `--plugin-dir` 方式。

### Q: 如何更新技能？

A: 运行 `install_patent_skill.py`，选择 [3] 更新技能。

### Q: 可以在多个项目中使用吗？

A: 可以！每个项目都需要单独安装一次。
